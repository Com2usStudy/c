# 자료형이란?
C에서 자료형은 변수나 데이터의 유형 또는 형식을 정의되는 개념이다.
자료형은 변수가 어떤 종류의 데이터를 저장하고 어떻게 저장할지를 결정한다.
C 언어에서 일반적으로 사용되는 자료형에는 다음과 같은 것들이 있다.

int (정수): 정수 값을 저장
크기: 대개 4바이트 (32비트 시스템) 또는 8바이트 (64비트 시스템)

float (부동 소수점): 단정도 부동 소수점 숫자를 저장
크기: 4바이트

double (더블 부동 소수점): 배정도 부동 소수점 숫자를 저장
크기: 8바이트

char (문자): 문자 하나를 저장
크기: 1바이트

short (짧은 정수): int보다 작은 범위의 정수를 저장
크기: 대개 2바이트

long (긴 정수): int보다 큰 범위의 정수를 저장
크기: 대개 4바이트 (32비트 시스템) 또는 8바이트 (64비트 시스템)

unsigned (부호 없는 정수): 음수를 표현하지 않는 정수를 저장
크기: 동일한 크기로 정수와 동일

bool (불리언): 불리언 값 (true 또는 false)를 저장
크기: 1바이트

enum (열거형): 명명된 정수 상수를 정의
크기: 대개 int와 동일

struct (구조체): 여러 다른 자료형을 하나로 묶어서 사용자 정의 데이터 구조를 정의
크기 : 구조체의 구성원에 따라 달라짐

union (공용체): 여러 다른 자료형 중 하나를 저장
크기 : 가장 큰 구성원의 크기와 동일합니다.
